You can try three main api through 3 steps:

step 1: preprocess paragraph
method:POST url: /KnowledgeExtraction/preprocess
data:A paragraph in Text format,such as:"2018-09-01周杰伦在鸟巢开演唱会。王菲献唱传奇。"
return: Tokenization of splited sentences of the paragraph.

step 2: preannotate preprocessed file
method:POST url: /KnowledgeExtraction/predict
data:Json string like response from step 1
return: 3 kinds(model,regular expression,dictionary) annotation results in one file

step 3: merge preannotated results
method:POST url: /KnowledgeExtraction/merge
data:Json string like response from step 2
return: merged annotation results

Besides,you can annotate sentence in text format:
post /KnowledgeExtraction/predict/sentence data:"2018-09-01周杰伦在鸟巢开演唱会。王菲献唱传奇。" in Text format

Also, you can try 3 kinds of pre annotation methods respectively
data: Json string like response from step 1
method:POST
1. model annotate. url:/KnowledgeExtraction/predict/model
2. dictionary annotate. url:/KnowledgeExtraction/predict/dict
3. regular expression annotate. url:/KnowledgeExtraction/predict/regex
